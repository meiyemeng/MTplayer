# MT播放器 v1.3.0

## 主要更新

- 网页、Windows、Android 手机、Android TV 与 macOS 统一为 v1.3.0。
- 配置源中的 `lives`、嵌套频道组、`proxy://`、M3U/M3U8/TXT 直播列表可自动识别并加入直播频道。
- 修复最后一个配置源无法彻底删除、同步后又恢复的问题。
- 后台记录最近登录 IP 与城市，并展示每位用户添加的配置源、影视接口、直播接口地址及数量。
- 新增普通用户、会员、VIP 等级管理，以及面向会员统一推送配置源和直播源的预留 API。
- 关于页加入 GitHub 仓库、全平台下载入口及支付宝自愿捐助二维码。
- Android 手机与 TV、macOS 直播页支持直接选择配置内频道播放。
- 新增数据库迁移，现有部署升级时自动保留原账号与同步数据。

## 部署提示

- 群晖 Docker 使用 `mtplayer/server:1.3.0-amd64`。
- Cloudflare Tunnel 无需开放群晖公网端口，源站继续使用 `http://mt-api:8080`。
- 建议在 Cloudflare 开启“添加访问者位置标头”，后台即可直接显示登录城市。
- 升级前请备份 PostgreSQL 数据和 `DATA_ENCRYPTION_KEY`。

## 隐私与内容说明

服务器只保存账号、设备、收藏、观看进度、设置和用户自行添加的地址，不预置、不存储影视内容。支付宝捐助完全自愿，不影响任何基础功能。
