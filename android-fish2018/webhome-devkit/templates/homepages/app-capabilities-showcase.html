WebHome SDK 能力测试台

真实资源与 App SDK 调用样例

此页用于开发者在 App WebHome 内直接测试开放能力。素材使用公开测试源；网盘结果通过 PanSou API 实时搜索并可交给 App 检测/播放。

网络与资源

演示 `fm.req()` 读取远端数据，`fm.res()` 为 DOM 图片/视频生成本地资源网关地址。

原生播放

演示 MP4、HLS、直播 HLS、临时多集和 artwork 预热。公开 HLS 不额外传 headers/cookies。

播放控制

控制当前 PlaybackService；没有播放服务时 Native 会安全返回空结果。

网盘搜索、检测与播放

默认搜索主文档示例关键词；也可载入内置网盘检测矩阵，覆盖 ok/bad/locked/unsupported 等状态样本。

App 入口

演示搜索、点播、收藏、直播、设置、历史记录、返回和刷新。

Chrome 与安全区

顶部安全区只在 edge/immersive/fullscreen 等融合状态启用，恢复 normal 后移除。

缓存

App 内使用 `fm.cache`；浏览器预览使用 localStorage fallback。

CSP VOD 调用

`fm.vod(siteKey, vodId, title, pic, { wallPic })` 依赖当前配置里的真实站点 key 和 vodId；这里保留可编辑字段。

如果不确定 vodId，可先用原生站点搜索进入详情，再从调试日志或站点 API 获取。
